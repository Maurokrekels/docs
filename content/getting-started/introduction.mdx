# Introduction

FiveGateway is a platform that lets you manage your FiveM server(s) from a central web dashboard. It gives you real-time insight into players, logs, statistics and more.

## What is FiveGateway?

FiveGateway consists of two parts:

1. **Web Dashboard** — Available at [my.fivegateway.com](https://my.fivegateway.com). This is where you manage your servers, view logs, monitor statistics and moderate players.
2. **FiveM Resource** — A resource you install on your FiveM server. It connects to the platform via WebSocket and sends real-time data.

## Features

- **Player Management** — View online players, search player history, kick players and take screenshots remotely.
- **Structured Logging** — Send custom logs from your scripts with searchable metadata.
- **txAdmin Integration** — Bans, warns, kicks and announcements from txAdmin are automatically logged.
- **Statistics** — Monitor player counts, uptime, CPU and memory usage.
- **Queue System** — Manage your server queue with priorities, whitelist mode and maintenance mode.
- **Resource Management** — Start, stop and restart resources on your server from the dashboard.
- **Auto-Update** — Update the FiveGateway resource directly from the dashboard without manual downloads.
- **Discord Bot** — Get notifications and run commands from Discord.
