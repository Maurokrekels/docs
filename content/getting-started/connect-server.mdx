# Connect Server

import { Callout, Steps } from 'nextra/components'

After creating an account you can connect your FiveM server to FiveGateway.

<Steps>

### Add a server in the dashboard

Go to the dashboard at [my.fivegateway.com](https://my.fivegateway.com) and click **Add Server**. Enter your server name and confirm.

### Copy your API key

After creating the server you receive a unique API key. You will need this to connect the FiveGateway resource on your server.

<Callout type="warning">
  Keep your API key safe and do not share it with others. Anyone with access to this key can send data to your dashboard.
</Callout>

### Install the resource

Install the FiveGateway resource on your FiveM server and enter the API key. See [Install Resource](/getting-started/install-resource) for the full installation instructions.

### Verify the connection

Once the resource is running and connected, your server will appear as **Online** in the dashboard. You can now start managing your server.

</Steps>

## Server Limit

The number of servers you can connect depends on your plan:

| Plan | Servers |
|------|---------|
| Starter | 1 |
| Pro | 3 |
| Premium | Unlimited |
