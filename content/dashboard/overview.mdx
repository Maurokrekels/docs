# Dashboard Overview

The FiveGateway dashboard at [my.fivegateway.com](https://my.fivegateway.com) is your central place for managing all your FiveM servers.

## Main Screen

After logging in you will see an overview of all your connected servers with their current status:

- **Online** — The server is connected and sending data.
- **Offline** — There is no active connection with the resource.

## Navigation

Use the sidebar to navigate to the different sections:

- **Players** — View and manage online and historical players.
- **Logging** — Real-time logs from your server.
- **Statistics** — Charts and metrics about your server.
- **Queue** — Configure and monitor the server queue.
- **Moderation** — Kick players and take screenshots remotely.
- **Resources** — Start, stop and restart resources on your server.
- **Settings** — API keys, plan and notifications.

## Switching Servers

If you have multiple servers connected (Pro or Premium plan), you can switch between servers using the dropdown menu at the top of the dashboard.

## Real-Time Data

The dashboard receives data via a WebSocket connection with the FiveGateway resource on your server. This means changes like players joining or leaving are visible immediately without refreshing the page.
