# Statistics

The statistics panel gives you visual insight into the performance and activity of your FiveM server.

## Available Metrics

- **Player Count** — Number of online players over time, with peaks and drops clearly visible.
- **Unique Players** — How many unique players visit your server daily, weekly or monthly.
- **Average Playtime** — How long players stay on your server on average per session.
- **Join/Leave Activity** — Overview of joins and leaves per time unit.
- **CPU Usage** — Server CPU usage percentage over time.
- **Memory Usage** — Server memory usage in MB over time.
- **Uptime** — How long the server has been running since the last restart.

## Time Period

Select the desired time period for the statistics:

- **24 hours** — Detailed overview of the past day.
- **7 days** — Weekly overview with daily trends.
- **30 days** — Monthly overview for long-term trends.

## Use Cases

Statistics help you recognize patterns, such as:

- At what times your server is busiest.
- Whether your player base is growing or shrinking.
- How long players stay on average.
- Whether server performance (CPU/memory) needs attention.

Use these insights to make decisions about server capacity, events and community management.
