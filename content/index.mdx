# FiveGateway Documentation

FiveGateway is a management platform for FiveM servers. Monitor players, view logs and track statistics from a single dashboard.

## Getting Started

New to FiveGateway? Follow these steps to get up and running:

1. [Create an account](https://my.fivegateway.com) — log in with Discord
2. [Add your server](/getting-started/connect-server) — click **Add Server** and copy your API key
3. [Install the resource](/getting-started/install-resource) — drop it in your `resources` folder and add your key to `server.cfg`

```cfg
set FIVEGATEWAY_API_KEY "your-api-key-here"
ensure fivegateway
```

Restart your server and it will appear as **Online** in the dashboard. See the [introduction](/getting-started/introduction) for a full overview.
