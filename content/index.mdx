## Quick Setup

Setting up FiveGateway takes just a few minutes:

<Steps>

### Create an account

Go to [my.fivegateway.com](https://my.fivegateway.com) and log in with Discord.

### Add your server

Click **Add Server** in the dashboard and copy your API key.

### Install the resource

Download the resource from the dashboard, place it in your `resources` folder and add this to your `server.cfg`:

```cfg
set FIVEGATEWAY_API_KEY "your-api-key-here"
ensure fivegateway
```

### Done

Restart your server. Once connected, your server appears as **Online** in the dashboard.

</Steps>

See [Install Resource](/getting-started/install-resource) for detailed instructions.
