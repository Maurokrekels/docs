# Changelog

Here you will find an overview of all updates and changes to the FiveGateway platform.

## v0.1.8-beta

*Current release*

- Web dashboard with server overview
- Real-time player list and player management
- Moderation tools: kick players and take screenshots from the dashboard
- Structured logging system with custom categories and searchable metadata
- txAdmin integration: automatic logging of bans, warns, kicks and announcements
- Statistics with charts for player counts, CPU, memory and uptime
- Queue system with priority roles, whitelist mode and maintenance mode
- Resource management: start, stop and restart resources from the dashboard
- Auto-update: update the FiveGateway resource directly from the dashboard
- FiveM resource with WebSocket connection
- Discord bot integration with notifications and slash commands
- Discord OAuth authentication
- Plans: Starter (free), Pro (€9/month), Premium (€19/month)
